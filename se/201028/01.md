```
此類題必考，要說明原因，說服助教就有分數
F：功能性需求
NF：非功能性需求
D：領域需求
```

```
GUI 以前幾乎沒有，所以以前是NF，後來都是標準為 F
即時生的軟體。是D，不同的系統，時間限制都不一樣。
電磁干擾標準：領域性需求。
資料庫連線，安全保護，幾乎都是功能性，以前是非功能性。
計算薪水，扣除是功能性
記憶體之需求（空間）非功能性
CMMI，公司有需求，為領域性

```
